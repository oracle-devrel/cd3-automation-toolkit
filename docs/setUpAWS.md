# **Manage DB@AWS using CD3 Automation Toolkit (COMING SOON)**
---

**CD3** extends its support to **Oracle DB@AWS**, making it easier to manage Oracle Databases on AWS.

With CD3, users can currently create and manage **ADB@AWS**. 

!!! info 
    Support will be extended to **Exadata@AWS** in future releases. 

To manage ADB@AWS, users should first complete the steps to <a href="../install-cd3"><u>Install the CD3 toolkit</u></a> and then <a href="../connect-container-to-aws-account"><u>Connect the CD3 container to AWS account</u></a> 

Once the CD3 container is successfully connected to AWS account, the Toolkit can be used to execute Create/Export workflows for ADB@AWS.

At present, CD3 for DB@AWS supports only CLI-based operations. Follow the instructions below to get started. 

1.  <a href = "../greenfield-aws-cli"><u>Create DB@AWS with CLI</u></a> 
2.  <a href = "../nongreenfield-aws-cli"><u>Export DB@AWS with CLI</u></a>
