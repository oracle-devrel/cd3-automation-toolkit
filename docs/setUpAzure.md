# **Set Up DB@Azure using CD3 Automation Toolkit**
---

**CD3** now extends its support to **Oracle DB@Azure**, making it easier to manage Oracle Databases on Azure.

With CD3, users can currently create and manage **ADB@Azure**. Support will be extended to **Exadata@Azure** in future releases. 

To set up ADB@Azure, users should first complete the steps to <a href="../install-cd3"><u>Install the CD3 toolkit</u></a> and then <a href="../connect-container-to-azure-subscription"><u>Connect the CD3 container to Azure subscription</u></a> 

Once the CD3 container is successfully connected to Azure subscription, the Toolkit can be used to execute Create/Export workflows for ADB@Azure.

At present, CD3 for DB@Azure supports only CLI-based operations. Follow the instructions below to get started.
